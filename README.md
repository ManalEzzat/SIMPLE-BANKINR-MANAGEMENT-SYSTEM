# SIMPLE-BANKINR-MANAGEMENT-SYSTEM
1. Overview
This project is l Console Application developed in C#. It simulates a real-world digital banking experience where
users can manage their finances through a secure and interactive interface. The program focuses on robust logic, data integrity, and user-friendly interaction.

In this project, I implemented:
Object-Oriented Programming (OOP)
Arrays for storing accounts data
Authentication logic
Input validation
Transaction handling logic
Loop-based menu system
